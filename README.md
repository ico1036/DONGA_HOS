# 🎮 게임 만들기 워크숍

아이들이 직접 만든 게임이 모이는 곳!

## 구조

```
/
├── index.html          # 메인 포털 (아이들 목록)
├── kids/
│   ├── index.json      # 아이들 목록 데이터
│   ├── {아이이름}/
│   │   ├── index.html  # 아이 개인 페이지 (게임 목록)
│   │   ├── games.json  # 게임 목록 데이터
│   │   └── {게임이름}/
│   │       └── index.html  # 실제 플레이 가능한 게임
```

Cloudflare Pages로 배포됩니다.
